Getting Started

Follow these steps to get your local AI coding agent up and running in minutes.

Prerequisites

Node.js & NPM

Required for the Web UI and backend server.

Python 3.x

Required for the CLI version of the agent.

Ollama

The engine that runs the LLMs locally.

Model

Recommended: qwen2.5-coder:7b

Installation Steps

1

Install Dependencies

Install the necessary Node packages for the web interface.

npm install
2

Prepare Ollama

Ensure Ollama is running and the model is downloaded.

ollama serve
ollama pull qwen2.5-coder:7b

How to Run

CLI Mode

The fastest way to start. No server required.

python cli/agent.py
  • • Use /model to switch LLMs
  • • Use /tools to see capabilities
  • • Type exit to quit

Web UI Mode

Full visual experience with real-time logs.

npm run start

Open your browser to:

http://localhost:5173
