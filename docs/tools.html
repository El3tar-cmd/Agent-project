Tool Reference

The agent has access to 24 tools that let it read, write, search, execute, archive, diff, lint, and call external APIs — fully autonomously.

📁 File System 💻 Shell / Code 🌐 Web / APIs 📊 Git 🗜️ Archive / Diff ✨ New in v2
📁 File System — Read, Write, Search
Tool Parameters Description
read_file path Read full file content with line numbers.
read_lines path, start, end Read a specific line range from a large file.
write_file path, content Create or overwrite a file with new content.
append_file path, content Append content to the end of a file.
replace_text path, old, new Targeted find-and-replace. Exact-match string substitution.
list_files path List all files and folders in a directory with icons.
search_in_files pattern, directory Case-insensitive recursive text search across all files.
grep pattern, path, context? Regex search with surrounding context lines.
create_dir path Create directory recursively (mkdir -p).
delete_file path Delete a file or directory. Requires confirmation in Web UI.
find_files new pattern?, directory, ext?, maxdepth? Find files by name pattern or extension. Skips node_modules/.git automatically. Returns up to 100 results.
💻 Shell & Code Execution
Tool Parameters Description
run_command command, background?, timeout? Execute any shell command. Optional background mode for long-running processes (servers, watchers).
python_eval code Execute Python code in-process. Useful for math, data processing, and quick scripts.
lint new file, linter? Run linters: ESLint for .js/.jsx/.ts/.tsx, flake8 for .py, JSON validator for .json, bash -n for .sh. Auto-detects from extension.
🌐 Web & External APIs
Tool Parameters Description
http_get url Fetch a URL and return cleaned text content. Strips HTML tags automatically.
http_post new url, body, method?, headers? POST/PUT/PATCH to any REST API with a JSON body and custom headers. Returns status code + response body.
search_web query Search DuckDuckGo and return the top results with titles, URLs, and snippets.
screenshot url_or_path, fullPage? Take a screenshot of a URL or local HTML file via Puppeteer. Saves as PNG.
🗜️ Archives & Diff
Tool Parameters Description
zip new action, file, source?, dest? create — zip or tar.gz an entire directory.
extract — unpack to a destination folder.
list — list archive contents without extracting.
diff_files new file1, file2 Unified diff between any two files. Reports lines added/removed. Identical files return a clear message.
📊 Git Operations
Tool Parameters Description
git_status — Show the current git status (modified, staged, untracked files).
git_diff file? Show unified diff for a specific file or all changes.
🧭 Navigation & Reasoning
Tool Parameters Description
cd path Change the agent's working directory. Persists across tool calls in the same session.
think thought Internal reasoning step with no external side effects. Used for planning before complex multi-step tasks.
ask_human question Pause execution and ask the user a clarifying question. Times out after 2 minutes.

Security Note

Tools like run_command, delete_file, and write_file have real side effects. The Web UI requires confirmation before delete_file runs. It is strongly recommended to run the agent in a sandboxed or development environment — never on a production machine with sensitive data.
